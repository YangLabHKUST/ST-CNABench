# cSCC Demo Fixture

This directory contains the real cSCC `P6_vis_rep1` demo fixture for ST-CNABench.

- It mirrors the real `P6_vis_rep1` SpaceRanger-style input bundle plus FACETS CNA ground truth.
- It is intended for public demo runs of data preparation, model execution, and `cna_profile` evaluation.
- It does not include subclone ground truth, so subclone-focused evaluation tasks are intentionally unsupported for this example.
- The full public dataset input contract is documented in [../../docs/data_preparation.md](../../docs/data_preparation.md).
